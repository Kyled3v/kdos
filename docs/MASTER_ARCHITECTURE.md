# KDOS v2 — Master Architecture

**Status:** Foundational document. This is the constitution of KDOS v2. All future decisions must align with the principles defined here.

---

## 1. Vision

- KDOS is the internal operating system for KyleDev.
- KyleDev is the public-facing agency.
- Clients never use KDOS.

---

## 2. Core Principles

- Business first.
- AI is the workforce.
- Workflow before UI.
- Production-ready code only.
- No technical debt.
- One file at a time.
- One feature at a time.

---

## 3. Development Workflow

- ChatGPT is the Lead Software Architect.
- Claude is the Implementation Engineer.
- VS Code is the development environment.
- Git commit after every completed capability.

---

## 4. AI Workforce Philosophy

- Every AI worker shares the same foundation.
- Employees communicate through workflows.
- Employees use tools rather than doing everything themselves.

---

## 5. Build Order

1. Foundation
2. Authentication
3. AI Workforce Engine
4. Executive Assistant
5. Software Engineer
6. KyleDev Integration
7. Business Modules
8. UI Polish

---

## 6. Coding Standards

- TypeScript strict mode.
- Single responsibility.
- Reusable code.
- No duplicate logic.
- No placeholder code.
- No unused files.
